# Auto Loot PoE Helper

Помощник авто-подбора лута для **Path of Exile 2**. Работает **только по скриншоту экрана**
(компьютерное зрение), без чтения памяти игры.

> ⚠️ Сторонняя автоматизация нарушает ToS GGG и может привести к бану аккаунта. Используешь на свой риск.
>
> ⚠️ Захват экрана видит игру только в режиме **Windowed** или **Windowed Fullscreen**.
> В Exclusive Fullscreen кадр будет пустым — переключи режим окна в настройках игры.

## Как это работает

1. В лут-фильтр (NeverSink) впрыскивается override-блок, который красит выбранные категории
   (валюта, фрагменты, гемы, вейстоуны) в уникальный **цвет-маркер** `[255, 0, 200]` (ярко-розовый/магента —
   такого цвета нет в окружении PoE2, нет ложных срабатываний).
2. Программа захватывает кадр окна игры и ищет этот цвет на экране.
3. При включённом режиме двигает мышь к ближайшей (или наиболее приоритетной) цели и кликает.
4. Движение мыши — геймерский стиль: быстрый прямой бросок к лейблу, приземление точно на цель.

## Что подбирается

Только «мелочёвка» — всё настраивается в `config/default.yaml → filter.categories`:
- **currency** — валюта (Chaos Orb, Divine Orb и т.д.)
- **fragments** — фрагменты и осколки
- **gems** — гемы (Uncut Skill/Support/Spirit Gem)
- **waystones** — вейстоуны

Уники, жёлтые вещи и мусор — **не подбираются**, берёшь сам.

**Порядок подбора:** сначала валюта (самая ценная), потом фрагменты, гемы, вейстоуны.
Внутри категории — ближайший к персонажу предмет. Настраивается через `loot.category_priority`.

## Режимы (loot.mode)

- **toggle** *(по умолчанию)* — F8 включает/выключает авто-подбор. Ничего держать не надо.
- **hold** — собирает, пока зажата клавиша `pickup`.
- **single** — один клик за нажатие `pickup`.
- **lazy** — подбор предмета **под курсором**: наводишь мышь на лут — он подбирается.

> Space в PoE2 = кувырок (dodge). Не назначай pickup на Space в hold-режиме.

## Хоткеи

| Клавиша | Действие |
|---------|----------|
| F8 | Мастер вкл/выкл (авто-подбор + авто-автоматика) |
| F7 | Переключить профиль (циклом) |
| F12 | Выход |

## Оверлей

Прозрачное click-through окно поверх игры показывает:
- Статус (ON / idle), режим, профиль
- Количество целей и целей в радиусе подбора
- Всего подобрано + статистика по категориям: `cur:47  frag:12  gem:3`
- HP% (если включена авто-фласка)

## Авто-фласка HP

Следит за орбом жизней. Когда HP падает ниже порога — нажимает клавишу фласки.
Самокалибровка: в первые 6 секунд запоминает максимум (= полный HP) и нормирует относительно него.

```yaml
hp_flask:
  enabled: true
  key: "1"
  threshold: 0.65   # нажать когда HP < 65%
  cooldown_ms: 4500
```

## Установка

```powershell
pip install -r requirements.txt
```

## Запуск

```powershell
# Двойной клик по run_calibrated.bat
# или вручную:
python -m src.main
python -m src.main --profile mapping
python -m src.main --no-overlay
python -m src.main --calibrate      # окно с подсветкой целей и маской
```

## Калибровка

```powershell
python -m src.calibrate
python -m src.calibrate --target myprofile
```

ЛКМ — взять цвет с экрана, Shift+ЛКМ — центр персонажа,
трекбары — допуск тона/насыщенности/яркости/площади/радиус.
`s` — сохранить профиль, `d` — сохранить debug-снимок в `_debug/`, `q` — выход.

## Профили

`config/profiles/<name>.yaml` — переопределяют значения из `default.yaml`.
В комплекте: `calibrated` (оптимальные настройки), `mapping` (большой радиус).

Смена на лету — F7 (циклом). Старт с профилем: `--profile calibrated`.

## Патч фильтра

```powershell
python -m src.core.filter_patcher --check    # статус
python -m src.core.filter_patcher --patch    # впрыснуть override (бэкап -> *.filter.bak)
python -m src.core.filter_patcher --unpatch  # удалить
```

После патча: Escape → Options → UI → перевыбрать фильтр.

## Авто-автоматика (по таймеру)

Секция `automation` в конфиге — авто-фласки/скиллы по интервалу (**по умолчанию выкл**).
Срабатывают только когда мастер F8 включён и окно игры в фокусе.
